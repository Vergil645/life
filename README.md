# life

## Description

Simple `C` implementation of Conway's Game of Life based on `SDL2` library.

## Build

```sh
cmake -S . -B build
makce -C build
```

## Run

```sh
./build/bin/start
```

## LLVM IR

Repository contains LLVM IR of `./src/app.c` file. It was generated by [genIR](./genIR) scipt.
